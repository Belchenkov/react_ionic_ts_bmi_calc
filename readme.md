# BMI Calc (React/Ionic/TS)

> Build App "BMI Calculator" usage React + Ionic 5 + Typescript

### [Demo](http://react-ionic-ts-bmi-calc-ab.surge.sh)

#### Get Started
```sh
   git clone 
```
```sh
   ionic serve 
```
or 
```sh
   npm start
```

> Run on Android
```sh
   ionic capacitor run android
```

> Run on IOS
```sh
   ionic capacitor run ios
```
